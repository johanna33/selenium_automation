Selenium Automation Framework

A functional, dynamic, modular, easy to handle web automation framework, created to conduct daily regression functional testing against web applications.

It provides supports for testing web based apps (multibrowsers) and mobile apps within same framework, with ease by only changing settings in property files.

Also supports execution in local machine or remote.

This framework is build to speed up the testing process among several end to end testing teams, with an amazing feature of being hook up easily on any web application given its modularity.


Technologies: Java SE 8, Selenium Webdriver 2.53.1, TestNg 6.9, Maven 3.5
